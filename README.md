# PhotoFrame


## 이 챕터를 통해 배운 것

- **Layout** 을 그리는 법
  - 가로 화면으로 그리기
- **Android Permission** 사용하기
- **View Animation** 사용하기
- **Activity Lifecycle** 알아보기
  - https://developer.android.com/guide/components/activities/activity-lifecycle?hl=ko
- Content Provider
  - SAF (Storage Access Framework)
